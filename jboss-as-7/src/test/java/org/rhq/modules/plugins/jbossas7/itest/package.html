This package contains the integration tests that run against JBoss AS 7 / EAP6 servers in standalone and domain mode. You can start the tests like this: mvn verify -DskipTests=false -Das7-itest=as7 -Das7.version=7.1.1.Final Subdirectories:
  • domain: test against a domain mode server
  • nonpc : tests that don't go through the full plugin-container/resource container; copied over from older integration tests
  • standalone: tests against a standalone server
See also this the wiki page at http://rhq-project.org/display/RHQ/Notes#Notes-IntegrationTests for more information
